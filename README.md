# TODO-List App

A simple and interactive To-Do List web application built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- Add new tasks
- Click on a task to highlight it
- Remove tasks with a delete button
- Responsive and clean user interface
